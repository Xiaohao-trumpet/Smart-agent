## Request Flow

```
┌──────────┐
│   User   │
└────┬─────┘
     │ 1. Type message
     ▼
┌─────────────────┐
│   OpenWebUI     │
└────┬────────────┘
     │ 2. POST /api/v1/chat
     │    {user_id, message}
     ▼
┌─────────────────────────────────────────────────────────┐
│  FastAPI                                                 │
│  ┌────────────────────────────────────────────────┐     │
│  │ 3. Validate Request (Pydantic)                 │     │
│  │ 4. Check Rate Limit                            │     │
│  │ 5. Get/Create Session                          │     │
│  └────────────────┬───────────────────────────────┘     │
└───────────────────┼─────────────────────────────────────┘
                    │ 6. Invoke Graph
                    ▼
┌─────────────────────────────────────────────────────────┐
│  LangGraph                                               │
│  ┌────────────────────────────────────────────────┐     │
│  │ 7. Execute model_node                          │     │
│  │    - Extract user_id, message from state      │     │
│  │    - Call UniversalChat.chat()                │     │
│  │    - Update state with response               │     │
│  └────────────────┬───────────────────────────────┘     │
└───────────────────┼─────────────────────────────────────┘
                    │ 8. Call model
                    ▼
┌─────────────────────────────────────────────────────────┐
│  UniversalChat                                           │
│  ┌────────────────────────────────────────────────┐     │
│  │ 9. Build messages list:                        │     │
│  │    - System prompt                             │     │
│  │    - Conversation history                      │     │
│  │    - New user message                          │     │
│  │ 10. Call OpenAI-compatible API                 │     │
│  │ 11. Update conversation history                │     │
│  └────────────────┬───────────────────────────────┘     │
└───────────────────┼─────────────────────────────────────┘
                    │ 12. API call
                    ▼
┌─────────────────────────────────────────────────────────┐
│  Model Backend (Qwen/Ollama/vLLM)                       │
│  ┌────────────────────────────────────────────────┐     │
│  │ 13. Generate response                          │     │
│  └────────────────┬───────────────────────────────┘     │
└───────────────────┼─────────────────────────────────────┘
                    │ 14. Return response
                    ▼
┌─────────────────────────────────────────────────────────┐
│  FastAPI                                                 │
│  ┌────────────────────────────────────────────────┐     │
│  │ 15. Calculate latency                          │     │
│  │ 16. Log request details                        │     │
│  │ 17. Return JSON response                       │     │
│  └────────────────┬───────────────────────────────┘     │
└───────────────────┼─────────────────────────────────────┘
                    │ 18. HTTP response
                    ▼
┌─────────────────┐
│   OpenWebUI     │
└────┬────────────┘
     │ 19. Display response
     ▼
┌──────────┐
│   User   │
└──────────┘
```

## Error Handling Flow

```
Request
  │
  ▼
┌─────────────────┐
│ Validation      │ ──► ValidationError ──► 422 Response
└────────┬────────┘
         │ Valid
         ▼
┌─────────────────┐
│ Rate Limiting   │ ──► RateLimitError ──► 429 Response
└────────┬────────┘
         │ OK
         ▼
┌─────────────────┐
│ Process Request │
└────────┬────────┘
         │
         ├──► ModelAPIException ──► 502 Response
         │
         ├──► SessionNotFound ──► 404 Response
         │
         ├──► Exception ──► 500 Response
         │
         ▼
┌─────────────────┐
│ Success         │ ──► 200 Response
└─────────────────┘
```
